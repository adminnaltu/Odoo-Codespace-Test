![jenkins](https://github.com/mohammedshahil/Odoo-Codespace/assets/33252126/dac25543-1429-4ae6-ab01-e3636e3b0429)

#### 3. Make the script executable
```
chmod +x odoo_codespace.sh

```
sudo ./odoo_codespace.sh
